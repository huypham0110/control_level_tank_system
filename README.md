# control_level_tank_system
## About this project
- This is a basic project, which is built to control the level of a single tank system.
- We used PID controller in Arduino Uno, design GUI with Winform C#
- Hardware:
  + Board Arduino Uno R3
  + Sensor Ultrasonic HR-SC04
  + Motor driver module L298N
  + Brushless motor submersible water pump AD20P-1230C
  + Power supply 12V 3.2A
